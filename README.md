
# 📚 StudyMate AI

## 🚀 AI-Powered Personal Study Assistant

StudyMate AI is an intelligent learning companion designed to help students study smarter.  
It provides features like doubt solving, quiz generation, study planning, and notes-based learning assistance.

The goal of this project is to create a simple AI-powered platform that improves student productivity and learning experience.


---

# ✨ Features

## 🤖 AI Doubt Solver
- Ask questions related to your study materials
- Get simple explanations
- Learn concepts faster


## 📝 Quiz Generator
- Generate practice questions
- Improve revision
- Test your knowledge


## 📅 Study Planner
- Create personalized study schedules
- Manage study hours
- Improve consistency


## 📚 Notes Management
- Read learning materials
- Process stored notes
- Use notes for better answers
